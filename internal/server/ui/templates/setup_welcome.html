Welcome to compliancekit

Five short steps and you'll have your first scan running.

  • 1 Pick a provider — we'll start with one and you can add more later in Settings → Providers.
  • 2 Drop in an API token (or connect via OIDC at v1.4 Phase 6). The daemon never logs the secret — it goes straight to the encrypted config column.
  • 3 We probe the provider's API once to confirm the token works. No mutating calls, no resources changed.
  • 4 Kick off the first scan. Results land in the dashboard within minutes for typical fleets.
No data leaves your daemon. No telemetry. No phone-home. Get started
